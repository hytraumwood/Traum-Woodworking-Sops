# Traum Tools Policy

- This SOP defines company policy on tools; which are provided by Traum, which are provided by the employee, and the policies regarding each

## Traum-Provided Tools
  - Traum will provide all corded tools required for a position's duties.
  - Traum will provide the following cordless tools in a kit for the employee/position;
    1. Circular Saw
    1. Router
    1. Sander
    - These tools are to be returned in like/similar condition upon termination of employment. Any misuse/unusual wear on Traum provided tools will be compensated by employee at return.
    - These tools are provided dependent on position

## Employee-Provided Tools
  - Employees are expected to provide their own cordless drills/drivers

## General Notes
  - Any bits/blades/pads required in the scope of work will be provided by Traum
  - Traum provided tools are Makita-branded; should an employee desire a tool to fit a different brand's battery, that tool must be provided by the **employee**
