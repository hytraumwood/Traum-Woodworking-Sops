# Hardware Standards

This SOP determines the hardware to be used unless otherwise specified by a project team.

See also: [Hinge Placement](hinge-placement.md) for centerline dimensions and hinge count guidelines.

## Cabinet Hinges

All cabinet hinges to be Salice brand; existing projects that used the Blum spec will continue.

### Full Overlay
- C2PBAE9
- w/ 2mm plate BAR3R29

### Half Overlay
- C2PBGE9
- w/ 2mm plate BAR3R29

### Inset
- C2PBPE9
- w/ 3mm plate BAR3R39

## Drawer Slides
- All drawer slides are also to be Salice brand where applicable

## Shelf Pins
- 5mm shelf pins
- Adj shelves drawn with three holes, 32mm spacing between each — one above shelf location, one below

## Shop Drawing Hardware Notes

- Hardware in the shop drawing must be verified by the engineer **before** reaching Stockbilling.

### Hardware Ordered by Traum
- All hardware to be ordered by Traum *must* appear in the **Hardware** tab in the CAD drawing.

### Hardware Excluded
- All hardware that is present but not to be ordered by Traum *must* appear in a tab **Hardware by Others** in the CAD drawing.
