
# SOP - Solid Wood Edge Application (Tongue vs Applied)

## PURPOSE

Define a clear standard for when solid wood edges are tongued vs. applied to ensure quality, alignment, and install efficiency.

## RESPONSIBILITY

* **Engineering/Drafting**: Define edge condition in drawings
* **Shop**: Execute per drawings and SOP
* **Project Management**: Ensure consistency and resolve exceptions

## Procedure

### 1. Standard Rule

1.1 Solid wood edges applied to panels (MDF/plywood) are **tongued by default**.

1.2 This includes:

* Drop edges
* Face frames
* Solid edges on panel products

1.3 Reason:

* Stronger joint
* Better alignment
* Easier installation
* Matches premium quality standards fileciteturn0file0

---

### 2. Face Frame Conditions

#### 2.1 Typical Condition

* Use **one fixed tongue + one sliding side** when spanning cabinets
* Provides alignment while allowing easier install

#### 2.2 Double Tongue

* Allowed when:

  * Members are wide enough (>1.5")
  * Tolerance is built in
* Avoid if fit-up becomes difficult

---

### 3. Prefinished Interiors

3.1 Use **one side tongued, one side loose**

3.2 Reason:

* Prevents damage to prefinished surfaces
* Avoids sanding/refinishing issues

---

### 4. When Applied (No Tongue) is Acceptable

Use applied edges when:

* Metal or decorative trim is involved
* Material layers interfere (e.g., cork)
* Finish sensitivity makes tongue risky
* Installation would be overly difficult with a tongue

---

### 5. Large or Tall Assemblies

5.1 Use:

* Single tongue + sliding cabinet (preferred), OR
* Double tongue with built-in tolerance

5.2 Add alignment features (blocking) when needed

---

### 6. Tolerances

6.1 Account for:

* Prefinished thickness variation
* Veneer stability

6.2 Prioritize:

* Clean exterior alignment
* Acceptable interior variation where hidden


## Expectation

* Tongue-and-groove is the **default standard**
* Exceptions must be intentional
* Drawings must clearly show non-standard conditions
