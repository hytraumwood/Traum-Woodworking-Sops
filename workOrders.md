# Work Order Standards

## Creating Work Orders

### Drafting
1. Go to the project page in Innergy. Select "Budget" tab.
1. The Budget\> Products page will automatically be open.
1. Select the products that pertain to the work order you are going to make (i.e., an entire room of scope items, just the base cabinets, all the doors, etc.)
1. Select "Create Work Order" just above the grid.
1. Name the work order, following the naming convention showed below
1. Select the "Drafting" Type, and set the critical submittal date (automatic dates are generated, but may not be accurate to the needs of the project)
1. Click the green button to create the work order
