# SOP- Item Sizing

## Entryway
  - All Shippable Items should be considered for entry into the residence, as well as the pathway for the specific room. Considerations to be made by the engineer with feedback from the field.

## Truck size

- The opening to the large shipping truck is 92" x 98"- anything larger than this opening should be split into two items

## Item size
- Any shippable item that is larger than 120" x 48", regardless of the combination of the dimensions, should be considered for size reduction for shipping and unloading.
- Full height cabinets w/o a toekick must be able to stand up in room, corner to corner, for install

## Item Weight
- Any shippable item that is calculated to weigh over 300 lb. should be considered for size reduction for shipping and unloading.
